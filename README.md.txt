# Controle de Gastos CLI

## Problema
Muitas pessoas têm dificuldade em controlar seus gastos diários, o que pode levar a desorganização financeira.

## Solução
Esta aplicação permite registrar, listar e calcular gastos de forma simples via linha de comando.

## Público-alvo
Pessoas que desejam controlar seus gastos de forma simples.

## Funcionalidades
- Adicionar gasto
- Listar gastos
- Calcular total

## Tecnologias
- Python
- Pytest
- Ruff

## Instalação
pip install -r requirements.txt

## Execução
python src/app.py add 50 mercado  
python src/app.py list  
python src/app.py total  

## Testes
pytest

## Lint
ruff check .
